Basic Message

https://github.com/.../features/0095-basic-message

All communication in DIDComm is asynchronous.
Here we show how we can send plain, signed and encrypted messages to the other party. 


    Nessus DIDComm CLI
    Version: ${nessusVersion}

    >> agent start

    >> wallet create --name Malathi

    Malathi>> protocol invitation receive --inviter=Government --url=http://${Host}/invitation?inviter=Government&method=${method}

    Malathi>> protocol basic-message send "This worked great. Looking forward to doing more of this exciting Trust over IP"

    Malathi>> protocol basic-message send --sign "Here a message that you can trust coming from me"

    Malathi>> protocol basic-message send --encrypt "Here a message that nobody else can see"

    Malathi>> message list
