Out-of-Band Invitation

https://github.com/.../features/0434-oob-invitation

One of the first things we might want to do, is to connect to one of the above institutions.
For example, we might want to connect to Government to get a verifiable credential that represents a digital passport.

Click on an 'invitation' link above, to see the out-of-band invitation messages generated by the respective institution's agent. Then use the Nessus CLI locally to receive that invitation and create a trusted connection from it.

Such a trusted peer to peer connection is the foundation for all further communication. 


    Nessus DIDComm CLI
    Version: ${nessusVersion}

    >> agent start

    >> wallet create --name Malathi

    Malathi>> protocol invitation receive --inviter=Government --url=http://${Host}/message/invitation?inviter=Government&method=${method}

    Malathi>> message list
